
Description:
RepeatCount program takes one input of string and another input of word.It counts how many matching words in the string
and prints the number of matching words

Setup instructions:
1.create a project folder
2.inside project folder
i.create app, web, src, tests, views---folders

ii.create composer.json file

go to your terminal

3.change your directory to project folder

4.run composer install

5.To run your tests
export PATH=$PATH:./vendor/bin
6.phpunit tests

7.To run your project in localhost
change your directory to web folder
php -S localhost:8000
